<div class="concept" id="pid">
    <div class="concept-header">
      <div class="concept-num">04</div>
      <div class="concept-title-block">
        <h2>PID 控制器 · PID Controller</h2>
        <div class="concept-subtitle">经典控制论 · 用于 Agent 输出质量反馈</div>
      </div>
      <span class="domain-badge tag-ctrl">控制</span>
    </div>

    <div class="block">
      <div class="block-label">核心公式</div>
      <div class="formula">u(t) = Kp·e(t)  +  Ki·∫e(t)dt  +  Kd·de/dt

e(t)    = 误差 = 目标分数 − 实际分数
Kp·e    = P 项：当前这次差多少 → 立即修正
Ki·∫e   = I 项：历史误差积累 → 系统性修正
Kd·de   = D 项：误差变化速度 → 预判式修正</div>
    </div>

    <div class="block">
      <div class="block-label">三项的工程映射</div>
      <table class="decision-table">
        <tr><th>PID 项</th><th>信号</th><th>Claude Code 中的动作</th></tr>
        <tr>
          <td><strong>P（当前误差）</strong></td>
          <td class="signal">这次输出得 4/10</td>
          <td class="action">立刻修改这次的 prompt 措辞/结构</td>
        </tr>
        <tr>
          <td><strong>I（历史积累）</strong></td>
          <td class="signal">最近 10 次平均 5.5/10</td>
          <td class="action">重写 system prompt，修改 CLAUDE.md 的 agent 规则</td>
        </tr>
        <tr>
          <td><strong>D（变化趋势）</strong></td>
          <td class="signal">得分从 7→6→5，趋势在恶化</td>
          <td class="action">预判性重构，不等质量崩溃再改</td>
        </tr>
      </table>
    </div>

    <div class="block">
      <div class="block-label">代码模式 — Agent 质量反馈系统</div>
      <div class="code-header">
        <span>quality-feedback.ts</span>
        <span>XerpaAI Agent</span>
      </div>
<pre><span class="kw">class</span> <span class="var">AgentQualityPID</span> {
  <span class="kw">private</span> history: <span class="kw">number</span>[] = [];
  <span class="kw">private</span> kp = <span class="num">0.5</span>, ki = <span class="num">0.1</span>, kd = <span class="num">0.3</span>;
  <span class="kw">private</span> target = <span class="num">8.5</span>;

  <span class="fn">record</span>(score: <span class="kw">number</span>): <span class="var">CorrectiveAction</span> {
    this.history.<span class="fn">push</span>(score);
    
    <span class="kw">const</span> e = this.target - score;                           <span class="cm">// 当前误差</span>
    <span class="kw">const</span> recentHistory = this.history.<span class="fn">slice</span>(-<span class="num">10</span>);
    <span class="kw">const</span> I = recentHistory.<span class="fn">reduce</span>((s, v) => s + (this.target - v), <span class="num">0</span>);
    <span class="kw">const</span> D = this.history.length > <span class="num">1</span>
      ? score - this.history[this.history.length - <span class="num">2</span>]  <span class="cm">// 正数=改善，负数=恶化</span>
      : <span class="num">0</span>;
    
    <span class="kw">const</span> signal = this.kp * e + this.ki * I - this.kd * D;

    <span class="cm">// 决策逻辑</span>
    <span class="kw">if</span> (Math.<span class="fn">abs</span>(e) > <span class="num">3</span>) <span class="kw">return</span> {
      type: <span class="str">'immediate'</span>,
      action: <span class="str">'rewrite_user_prompt'</span>,    <span class="cm">// P 大：立即修这次</span>
      reason: <span class="str">`Score ${score} vs target ${this.target}`</span>
    };

    <span class="kw">if</span> (I > <span class="num">8</span>) <span class="kw">return</span> {
      type: <span class="str">'systemic'</span>,
      action: <span class="str">'update_system_prompt'</span>,    <span class="cm">// I 大：系统性问题</span>
      reason: <span class="str">`Accumulated error: ${I.toFixed(1)}`</span>
    };

    <span class="kw">if</span> (D &lt; -<span class="num">1</span>) <span class="kw">return</span> {
      type: <span class="str">'preemptive'</span>,
      action: <span class="str">'restructure_agent_pipeline'</span>, <span class="cm">// D 负：趋势恶化</span>
      reason: <span class="str">`Quality trending down: ${D.toFixed(2)}/call`</span>
    };

    <span class="kw">return</span> { type: <span class="str">'none'</span>, action: <span class="str">'continue'</span> };
  }
}

<span class="cm">// 用法：在 agent 每次执行后记录</span>
<span class="kw">const</span> pid = <span class="kw">new</span> <span class="var">AgentQualityPID</span>();
<span class="kw">const</span> result = <span class="kw">await</span> agent.<span class="fn">run</span>(task);
<span class="kw">const</span> action = pid.<span class="fn">record</span>(<span class="kw">await</span> <span class="fn">evaluate</span>(result));
<span class="kw">if</span> (action.type !== <span class="str">'none'</span>) <span class="fn">handleCorrection</span>(action);</pre>

    <div class="callout callout-warn">
      <strong>⚠ 反模式（只看 P）</strong>
      每次输出差了就改措辞，从不看历史趋势。结果是过度修正——像没有 D 项的控制器，会在目标附近震荡，无法稳定收敛。
    </div>

    <div class="quick-ref">
      <h4>调参直觉</h4>
      <ul>
        <li>Kp 太高：修正过猛，质量在目标附近上下震荡</li>
        <li>Ki 太高：对历史误差太敏感，频繁触发"系统性重写"</li>
        <li>Kd 太高：对趋势过于敏感，还没到问题就开始重构</li>
        <li>推荐起点：Kp=0.5, Ki=0.1, Kd=0.3，根据实际震荡情况调整</li>
      </ul>
