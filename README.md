## 这是什么

一个面向工程师的 Claude Code 知识社群，包含：

- **Claude Code 日报** — 每日动态、新功能、最佳实践
- **心智模型手册** — 用物理/数学/控制论的框架理解 Claude Code，而不是靠直觉瞎调
- **实战模板库** — 可直接复用的 SOUL.md / SKILL.md / agent 配置
- **社区讨论** — 提问、分享、案例

## 心智模型手册（Vol.01）

把 7 个来自物理学、数学、控制论的核心概念，映射到 Claude Code 真实开发场景：

| 概念 | 用在哪 |
|------|--------|
| 模拟退火 | Prompt 调优卡住时如何跳出局部最优 |
| 傅里叶分解 | Multi-agent 任务拆解与正交测试 |
| 相变 | Context/规则加多了系统突然变差的临界点 |
| PID 控制器 | Agent 质量反馈的系统性调节 |
| 奇异吸引子 | System Prompt 用边界约束而非正向指令 |
| 香农熵 | Prompt 歧义诊断与稳定性 |
| 帕累托前沿 | 速度/质量/成本三角的架构决策 |
