大多数人只用 Claude Code 的对话框，不知道有这些快捷键：

**对话相关：**

| 快捷键 | 功能 |
|---|---|
| `↑` / `↓` | 浏览历史命令 |
| `Ctrl+C` | 中断当前执行 |
| `Ctrl+L` | 清空屏幕（保留上下文） |
| `Ctrl+R` | 搜索历史命令 |
| `Tab` | 自动补全文件路径 |

**Slash 命令：**

| 命令 | 功能 |
|---|---|
| `/clear` | 清空对话历史（重置上下文） |
| `/compact` | 压缩上下文（保留摘要） |
| `/help` | 查看所有可用命令 |
| `/model` | 切换模型 |
| `/cost` | 查看本次 session 费用 |
| `/plan` | 进入 Plan Mode |

**最实用的两个：**

**`/clear` vs `/compact`**

`/clear` = 完全重置，新对话
`/compact` = 保留摘要，继续对话

长任务中途需要清理上下文但不想从头开始 → 用 `/compact`
任务完成想开新任务 → 用 `/clear`

**`Ctrl+C` 中断执行**

Claude Code 在执行长任务时，你可以随时 `Ctrl+C` 打断它，然后补充新指令。不要等它跑完再纠正方向，越早干预越好。

养成这些习惯，Claude Code 的操作效率会提升一个档次。
