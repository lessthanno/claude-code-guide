Claude Code Pro 订阅是固定费用，但如果你用 API Key 模式，成本管理就很重要。

**查看当前消耗：**

```bash
/cost
```

这会显示当前 session 消耗的 token 数量和估计费用。养成习惯，每次长任务结束后看一眼。

**高费用操作清单（要小心）：**

1. **把整个代码库扔进上下文** — 每次对话都 Read 所有文件，费用快速累积
2. **不压缩的长对话** — 每次对话都携带全部历史
3. **没有目的地让 Claude 探索** — "帮我看看这个项目有什么问题"
4. **频繁用 Opus 做简单任务** — Opus 适合复杂推理，不是所有任务都需要

**省钱策略：**

**精准定位，不要泛探索**
```bash
# 贵：让 Claude 自己找
"帮我找一下 payment 相关的 bug"

# 便宜：先精准定位
"读 /src/services/payment.go 第 145 行，分析这个函数的问题"
```

**合理选择模型**

| 任务类型 | 推荐模型 |
|---|---|
| 复杂架构设计 | Opus |
| 日常代码编写 | Sonnet |
| 格式化/简单修改 | Haiku |

**用 CLAUDE.md 限制探索范围**

```markdown
## 不读以下目录（节省 token）
- /vendor/, /node_modules/
- /docs/（除非明确要求）
```

**设置预算限制（API 用户）**

在 Anthropic Console 里设置月度预算上限，超出后自动停止请求。别让账单超出预期。

成本管理不是省钱，是让每个 token 都有价值。
