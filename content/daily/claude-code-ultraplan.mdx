Claude Code 最新版本加入了 `/ultraplan` 命令。核心逻辑：

- 任务规划阶段在云端完成（利用更大上下文窗口）
- 执行计划传回本地终端逐步执行
- 适合超大型重构任务，不再受本地上下文限制

**实际场景**：跨文件重构、大型 PRD 实现、多模块联动修改。

```bash
/ultraplan "重构整个认证模块，从 JWT 迁移到 Session-based"
```

计划会列出所有文件修改点，确认后才开始执行。
